# Courses Platform (Django LMS)

A Learning Management System (LMS) built using Django to manage courses, students, and academic interaction.

## 🚀 Features
- User authentication (Admin, Student, Advisor)
- Course management system
- Course enrollment
- Project tracking and advisor feedback
- Interactive dashboards

## 🛠️ Technologies Used
- Python
- Django
- SQLite
- HTML, CSS

## ⚙️ Installation

```bash
git clone https://github.com/mqudah78/Courses-platform.git
cd Courses-platform
pip install -r requirements.txt
python manage.py runserver
